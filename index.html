Política de Privacidad

Última actualización: 19 de febrero de 2025.

1. Información que recopilamos

Este bot de Discord ofrece las siguientes funciones:

  • Comprobar si un servidor de Minecraft está activo.
  • Proporcionar el enlace a nuestra página web.
  • Mostrar información del servidor.

2. Uso de la información

Los datos recopilados se utilizan únicamente para responder a comandos dentro de Discord. No compartimos información con terceros.

3. Seguridad

Aplicamos medidas de seguridad para proteger el bot contra accesos no autorizados o modificaciones indebidas.

4. Tienda de Discord

El bot ofrece acceso a una tienda de Discord sin almacenar información personal del usuario.
